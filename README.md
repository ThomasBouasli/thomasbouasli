## 📚 Bibliografia

Livros que li da área de TI

### Clean Code

> Não acho que precise de muita explicação, foi escrito por Uncle Bob e é mundialmente reconhecido. Define boas práticas durante o desenvolvimento de um software, 
> seguindo os princípios ágeis, afinal, Uncle Bob é um dos criadores.


### Nature of Code

> Livro de Daniel Shiffman, alguém que me inspirou a ver programação com outros olhos. O livro é sobre a aplicação de fenómenos naturais em um cenário artificial 
> com código, por exemplo forças, colisões, e até mesmo redes neurais artificiais. Não terminei sua leitura, mas já acompanhei todos os exemplos em seus vídeos 
> complementares.

### Make your own neural network

> Esse é bem auto explicativo, nele eu aprendi o funcionamento, o desenvolvimento e a aplicação de redes neurais feitas do zero, com nenhuma dependência, usando 
> python. Não é a forma mais prática, mas antes de abstrair uma ferramenta, é bom saber que problema ela tenta solucionar.
